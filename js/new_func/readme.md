- 作用域链
定义：作用域的集合就是作用域链
1、函数在执行的过程中，先从自己内部寻找变量
2、如果找不到，再从创建当前函数所在的作用域去找，从此往上，也就是向上一级找。
当在作用域内访问 变量/方法 的时候，会找离自己最近的那个 变量/方法 （就近原则）
