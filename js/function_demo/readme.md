- this
    函数运行时创建的一个指针    指向谁呢 ？
       1. 指向是不固定的    由运行的方式决定
       2.  add 作为普通函数被调用的时候， this指向？

            window

       3. 作为对象的方法调用的时候，this指向对象 
       4. 函数作为构造函数使用的时候    this指向实例
       5. 函数作为事件的回调函数来执行的时候  this 指向事件发生的元素
