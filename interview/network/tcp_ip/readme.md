# 三次握手 四次挥手
    - 三次握手 建立链接
        - 确认 双方都有发送及接收的能力
    - 四次挥手 断开链接

- B/S架构
    - 只需要一个URL，不要安装

- C/S架构 
    - 要适配客户端 做很多套   微软、linux、mac 不一样
    - JAVA    htmljscss就开始起来了

- BS CS 底层通信都是TCP/IP

- TCP/IP
    - 七层 五层协议
    - IP 唯一性
    - TCP/UDP  确保数据传输正确且完整
    - HTTP  应用层协议 基于TCP/IP
    - HTTP 是无状态的，通信完会关闭
